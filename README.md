# ECG Denoising and Audio Equalizer using MATLAB

A Digital Signal Processing project implemented in MATLAB focusing on ECG signal denoising and audio equalizer design using FIR and IIR filters.

## Project Overview
This project applies DSP concepts to real engineering problems including biomedical signal denoising and audio frequency filtering.

The project includes:
- ECG signal denoising
- FIR and IIR filter implementation
- FFT analysis
- Frequency-domain filtering
- Audio equalizer design
- Signal visualization and comparison

## Features
- ECG noise reduction using digital filters
- FIR and IIR filter design and comparison
- Audio equalizer implementation
- Frequency spectrum analysis using FFT
- Time-domain and frequency-domain visualization
- Signal filtering and reconstruction

## Technologies Used
- MATLAB
- Digital Signal Processing (DSP)
- FIR Filters
- IIR Filters
- FFT Analysis
- Signal Processing Toolbox

## Files
- `DSP_final_project_1.m` - ECG denoising implementation
- `DSP_final_project_2.m` - Audio equalizer implementation
- `DSP.project.pdf` - Full project report and analysis

## Concepts Covered
- Digital Filtering
- Frequency Response
- FFT and FFT Shift
- Noise Reduction
- Signal Reconstruction
- Biomedical Signal Processing
- Audio Signal Processing

## How to Run
1. Open MATLAB.
2. Open the project folder.
3. Run the required `.m` file:
   - `DSP_final_project_1.m`
   - `DSP_final_project_2.m`
4. View generated figures and signal analysis results.
